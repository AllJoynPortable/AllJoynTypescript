Welcome to AllJoyn TypeScript HTML Application!

AJ.TS HTML Application is both, a sample application utilising AJ.TS base implementation and a toolset enabling instant creation and testing of AllJoyn Devices.

Our online tools allow you to bootstrap one of selected boards, so they can be controller over the air, create and control your devices. You can also find loads of online device samples and demos.

  • Bootstraping instructions for several boards, so the code can be updated over-the-air later (in preparation)
  • Creating AllJoyn devices on-the-fly in the browser
  • Testing AllJoyn devices on-the-fly in the browser
  • Generating TypeScript / JavaScript / C# / ANSI C code (in preparation)
  • Exploring AllJoyn devices connected to AllJoyn Router (browser based AllJoyn Explorer) (in preparation)
  • AllJoyn protocol analysys tool
  • Routing messages via Azure IoT Hub (in progress)
