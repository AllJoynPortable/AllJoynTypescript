WebSocket --> Socket Proxy

AllJoyn TypeScript HTML implementation uses WebSocket to connect to AllJoyn. As AllJoyn Router currently doesn’t support WebSocket connection it’s necessary to use a proxy. One possibility is websockify which can be downloaded from here:

https://github.com/kanaka/websockify

Before starting AllJoyn TS HTML demo, you need to start websockify from command line as follows:



After proxy is started, make sure you have correct settings in Setup menu.
